## Student Grade Tracker (Java)

The **Student Grade Tracker** is a Java-based application designed to efficiently manage and analyze student academic performance. This project allows users to input, store, and process student grades using arrays or ArrayLists, making it ideal for learning fundamental data structures and basic Java programming concepts.

###  Features

* Input and manage multiple student grades
* Calculate:

  * Average score
  * Highest score
  * Lowest score
* Store data using arrays or ArrayLists
* Generate a clear summary report of all student performances
* Console-based interface (can be extended to GUI)

### Technologies Used

* Java
* Core Java concepts (loops, conditionals, collections)

### Purpose

This project is aimed at beginners to strengthen their understanding of:

* Data handling in Java
* Basic algorithms for calculations
* User interaction through console applications

### Future Enhancements

* Add GUI using Swing or JavaFX
* Store data in files or databases
* Include student names and IDs
* Add grade classification (A, B, C, etc.)
